# siftrate

This site was made more for myself, I always wanted a place where you can find a movie / series / cartoon to rate and comment, and finally a mini-dream came true.
Technology stack: T3 Stack, SCSS module

## Run Locally

Clone the project

```bash
  git clone https://github.com/sifmeop/siftrate.git
```

Go to the project directory

```bash
  cd siftrate
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
